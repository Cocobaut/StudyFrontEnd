Yêu cầu: Tạo 1 thư mục đặt tên là css, trong thư mục css tạo file styles.css, sau đó trong file index.html hãy link thêm file stle.cc vào (dạng đường dẫn relative-tương đối). Tạo ra các đoạn văn như dưới đây

Tạo thẻ p như này và thêm class="color-red", sau đó thêm css để dòng chữ này thành màu đỏ (Dùng bộ chọn class - class selector)

Tạo thẻ p như này và thêm class="color-green", sau đó thêm css để dòng chữ này thành màu xanh (Dùng bộ chọn class - class selector)

Câu 2: Bài tập về background

Yêu cầu 1: Tạo 1 thẻ div, đặt tên class cho thẻ div này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 200px, chiều cao 100px. Có background màu xanh lá như hình dưới đây

Yêu cầu 2: Tạo 1 thẻ div, đặt tên class cho thẻ div này tùy ý, sau đó viết css theo tên class đã đặt. Chiều rộng 100%, chiều cao 200px. Có bachground là hình ảnh, khi scroll hình ảnh đứng tên. Làm như dưới đây
